Show or create the screenplay structure as a beat sheet. Argument: $ARGUMENTS (optional — «create» to generate placeholder scenes).

## Режим 1 — Показать (по умолчанию)

1. Прочитай CLAUDE.md — тип проекта, структурная модель, структурные точки
2. Прочитай все файлы в scenes/ — извлеки заголовки и ключевые события
3. Прочитай analytics/compass_artifact.md (если есть) — жанровые системы
4. Выведи аутлайн:

```
## Аутлайн: НАЗВАНИЕ

### Акт I (сцены 01-05)
01. Название — ключевое событие [✓ написано / ✗ пусто]
02. Название — ключевое событие [✓]
...

### Акт II (сцены 06-15)
...

### Акт III (сцены 16-20)
...

## Структурные точки
- Инцидент: сцена 03 ✓
- Поворот 1: сцена 05 ✓
- Середина: — ✗
- Поворот 2: — ✗
- Кульминация: — ✗

## Пробелы
- Между сценами 08 и 09 нет связки
- Нет сцены для [структурная точка]
```

5. Предложи что добавить или переставить

## Режим 2 — Создать placeholder-сцены

Если аргумент содержит «создать» или передан список строк вида:
```
01 Название сцены
02 Другое название
...
```

1. Для каждой строки создай файл `scenes/NN_название.md` с минимальным содержимым:
   - Художественный: заголовок + пустой slug line + комментарий `<!-- TODO -->`
   - Документальный: заголовок + пустая таблица VIDEO/AUDIO + комментарий
2. Обнови таблицу сцен в CLAUDE.md (статус: план)
3. Добавь запись в «Историю изменений»

## Итог

Скажи пользователю что делать дальше: `/new-scene NN Название` для написания конкретной сцены.

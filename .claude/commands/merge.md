Merge two scenes/blocks into one. Argument: $ARGUMENTS (two numbers, e.g. `05 06`).

## Шаг 1 — Прочитай обе сцены

1. Прочитай CLAUDE.md — определи тип проекта
2. Найди оба файла в scenes/ по номерам
3. Прочитай содержимое обоих файлов целиком
4. Покажи пользователю краткое содержание обоих и попроси подтверждение

## Шаг 2 — Объедини

Первый файл (меньший номер) становится объединённым:

### Художественный
- Оставь заголовок первой сцены (или предложи новый)
- Объедини действие и диалоги: содержимое второй сцены добавляется после первой
- Если у второй сцены другая локация — добавь новый slug line внутри файла

### Документальный
- Оставь заголовок первого блока (или предложи новый)
- Объедини таблицы: строки второй таблицы добавляются после первой
- Если разные темы — добавь визуальный разделитель `---` между ними

## Шаг 3 — Удали второй файл

Удали файл с бОльшим номером.

## Шаг 4 — Перенумеруй последующие

Если есть файлы с номерами выше удалённого — перенумеруй:
- Двухфазный rename через temp-имена
- Обнови заголовки внутри файлов

## Шаг 5 — Обнови CLAUDE.md

- Обнови таблицу сцен (объединённая строка + перенумерация)
- Добавь запись в «Историю изменений»

## Шаг 6 — Итог

Выведи результат: что объединено, какие файлы перенумерованы.

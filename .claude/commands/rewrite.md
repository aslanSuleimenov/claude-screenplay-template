Rewrite a scene/block based on notes. Argument: $ARGUMENTS (scene number and notes, e.g. `05 sharpen conflict, cut exposition`).

## Шаг 1 — Подготовка

1. Прочитай CLAUDE.md — тип проекта, персонажи, структура
2. Найди файл в scenes/ по номеру
3. Прочитай сцену целиком
4. Прочитай analytics/compass_artifact.md — жанровые системы
5. Прочитай analytics/avoid-ai-writing-tells.md — запрещённые паттерны

## Шаг 2 — Анализ замечаний

Разбери заметки из аргумента. Типичные запросы:
- «усилить конфликт» — поднять ставки, добавить противодействие
- «убрать экспозицию» — показать действием, не рассказывать
- «сократить» — убрать повторы, ужать до сути
- «добавить подтекст» — то, что не говорится вслух
- «переписать диалог» — сделать живее, индивидуальнее
- Любые другие конкретные замечания

## Шаг 3 — Перепиши

Перепиши сцену с учётом:
- Замечаний пользователя (главный приоритет)
- Жанровых систем из compass
- Запрещённых AI-паттернов из avoid-ai-writing-tells.md
- Формата сцены по типу проекта (художественный/документальный)

Сохрани в тот же файл.

## Шаг 4 — Покажи diff

Выведи ключевые изменения: что убрано, что добавлено, что переработано.

## Шаг 5 — Обнови CLAUDE.md

- Добавь запись в «Историю изменений»
- Если изменилось ключевое событие сцены — обнови таблицу сцен

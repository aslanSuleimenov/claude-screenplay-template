Check project status and output diagnostics.

1. Прочитай CLAUDE.md — извлеки:
   - Поле «Тип» (художественный / документальный / вертикальная микродрама)
   - Название, жанр, логлайн
   - Количество строк в таблице сцен

2. Посчитай файлы в scenes/ (*.md, не считая 00_*)

3. Проверь наличие ключевых файлов:
   - analytics/compass_artifact.md
   - analytics/avoid-ai-writing-tells.md
   - memory/session_log.md

4. Выведи сводку:

```
Проект: [Название]
Тип: [тип]  ← если пустой: ⚠ НЕ УСТАНОВЛЕН → запусти /startproject
Жанр: [жанр]
Логлайн: [первые 80 символов]

Сцен в scenes/: N
Compass: [есть / нет → запусти /compass]
Session log: [есть / нет]

Готов к работе: ДА / НЕТ
```

Если «Тип» пустой или не установлен — это единственная причина вывести «НЕТ».
Если compass_artifact.md отсутствует — предупредить, но не блокировать.

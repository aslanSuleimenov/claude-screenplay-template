Delete a scene/block and renumber following ones. Argument: $ARGUMENTS (scene number).

## Шаг 1 — Найди файл

1. Прочитай CLAUDE.md — определи тип проекта
2. Найди файл в scenes/ по номеру (NN_*.md)
3. Покажи пользователю содержимое файла (первые 10 строк) и попроси подтверждение
4. До удаления — создай резервную копию файла в `versions/backup_ГГГГММДД_ЧЧММ_удаление_NN.md`

## Шаг 2 — Удали файл

После подтверждения — удали файл.

## Шаг 3 — Перенумеруй последующие

Если есть файлы с номерами выше удалённого — перенумеруй их:
1. Двухфазный rename через temp-имена (избежать коллизий на Windows)
2. Обнови заголовки внутри файлов:
   - Художественный: `# Сцена XX:` → `# Сцена NN:`
   - Документальный: `# Блок XX:` → `# Блок NN:`

## Шаг 4 — Обнови CLAUDE.md

- Удали строку из таблицы сцен
- Обнови номера в оставшихся строках таблицы
- Добавь запись в «Историю изменений»

## Шаг 5 — Итог

Выведи что удалено и какие файлы перенумерованы.

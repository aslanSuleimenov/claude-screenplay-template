Create a new scene/block. Argument: $ARGUMENTS — number and title, e.g. `05 Bar` or `05: Mountains`.

1. Прочитай CLAUDE.md — определи **тип проекта** (художественный или документальный), персонажей, структуру, стиль.
   Если поле «Тип» пустое или не установлено — остановись и скажи: «Тип проекта не определён. Запусти /startproject чтобы настроить проект.»
2. Проверь наличие analytics/compass_artifact.md:
   - Если существует — прочитай (жанровые системы помогут написать сцену в стиле жанра)
   - Если нет — продолжи без него, но рекомендуй: «Запусти /compass для создания жанрового справочника»
3. Разбери аргумент: первое число — номер, остальное — название
4. Если номер занят или вставляется между существующими — предупреди и попроси подтверждение перед перенумерацией. Перенумерацию делай двухфазно (сначала temp-имена, потом финальные)

## Если тип = художественный

5. Напиши сцену строго по правилам converter_MD_DOCX/README.md (раздел «Художественный формат»):
   - `# Сцена NN: Название`
   - `**ИНТ./НАТ. ЛОКАЦИЯ — ВРЕМЯ**`
   - Имена персонажей **ЖИРНЫМИ ЗАГЛАВНЫМИ**
   - Одна пустая строка между блоками, ноль — между именем и диалогом

## Если тип = документальный

5. Напиши блок строго по правилам converter_MD_DOCX/README.md (раздел «Документальный формат»):
   - `# Блок NN: Название`
   - Таблица `| VIDEO | AUDIO |`
   - **V/O:** для закадрового текста, **SOT ИМЯ:** для интервью
   - *(NAT SOUND: ...)*, *(МУЗЫКА: ...)*, *(SFX: ...)* курсивом
   - **SUPER:** для титров, B-ROLL: для перебивок
   - Размеры планов: WS, MS, CU, ECU, AERIAL

## Общее для обоих типов

6. Проверь по analytics/avoid-ai-writing-tells.md — нет ли запрещённых паттернов
7. Сохрани как `scenes/NN_название.md`
8. Обнови таблицу сцен и историю изменений в CLAUDE.md

Research a topic for the screenplay. Argument: $ARGUMENTS (research topic).

1. Read CLAUDE.md — type, genre, setting, period, Seed
2. Search online (WebSearch) — run 2–3 searches with different angles on the topic:
   - Facts, statistics, verified data
   - Specific vocabulary, jargon, insider language
   - Visual and sensory details: what it looks like, sounds like, smells like
   - Unexpected or counterintuitive details — what most people don't know about this world

3. If the topic relates to a profession — find: daily routine, tools, hierarchy, slang, what goes wrong
4. If the topic relates to a location — find: geography, microculture, class dynamics, what outsiders miss
5. If the topic relates to a historical period — find: specific objects, prices, constraints, what people worried about

6. Filter by project type:

**Fiction:** prioritize sensory detail and behavioral specifics. What would a character in this world say, carry, notice? What would they never do?

**Documentary:** prioritize real people, real cases, statistics with a human face. What makes a good SOT (sound on tape)? What visual evidence exists? What is contested or unresolved?

7. Check the result before saving — cut anything that sounds like a summary or a textbook. Keep only what could appear in a scene.

8. Save to `analytics/research_[topic].md`:

```
## Research: [topic]

Date: [today]
Project: [title]

### Hard facts
(verified, specific, with numbers where possible)

### Sensory and behavioral details
(what to put in action lines, props, costume, dialogue texture)

### Documentary material
(for doc projects: real cases, potential SOT subjects, visual evidence — skip for fiction)

### What not to use
(clichés about this world — the obvious version everyone already knows)

### Sources
- [links]
```

9. Cross-reference with scenes/ — name 2–3 specific scenes where these details could land.

Analyze a scene. Argument: $ARGUMENTS (scene number or title).

1. Найди файл в scenes/ по номеру или названию
2. Прочитай сцену целиком
3. Проверь наличие analytics/compass_artifact.md:
   - Если файл существует — прочитай его (жанровые системы)
   - Если файла нет — предупреди: «compass_artifact.md не найден. Запусти /compass [жанр — логлайн] чтобы создать жанровый справочник. Анализ продолжится без жанровых систем.»
4. Дай оценку по каждой системе из compass (1-10), если compass_artifact.md есть:
   - Переворот заряда (+ → - или - → +)?
   - Ставки растут?
   - Визуальный образ без слов?
   - Двигатель истории работает?
5. Проверь по analytics/avoid-ai-writing-tells.md — нет ли запрещённых паттернов (включая русские AI-маркеры: «данный», «осуществлять», «является», правило трёх, эмоциональные плейсхолдеры)
6. Укажи конкретные проблемные строки
7. Предложи правки в формате БЫЛО → СТАЛО
8. Ответь на вопрос: можно ли эту сцену вырезать без потери для истории? Если да — что именно потеряется, и можно ли это передать иначе?

Ничего не сохраняй. Только анализ и рекомендации.

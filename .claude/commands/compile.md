Compile the screenplay into DOCX.

1. Прочитай CLAUDE.md — проверь поле «Тип». Если пустое — остановись: «Тип проекта не определён. Запусти /startproject.»
2. Проверь scenes/ — выведи список всех файлов по порядку
3. Проверь на проблемы:
   - Пустые файлы (0 строк контента)
   - Файлы без заголовка `# Сцена` / `# Блок`
   - Пропуски в нумерации (например: 01, 02, 04 — нет 03)
   - Если есть проблемы — выведи их и спроси продолжать ли
4. Запусти конвертер:
   ```
   python converter_MD_DOCX/md_to_docx.py
   ```
5. Выведи результат: путь к файлу, количество сцен, количество параграфов
6. Если есть ошибки — покажи их и предложи исправление

Если ошибка `ModuleNotFoundError: No module named 'docx'`:
```
pip install python-docx
```
(Именно `python-docx`, не `docx` — это разные пакеты.)

Конвертер читает все .md из scenes/ по алфавиту автоматически. Файл 00_* идёт как титульная страница.

Build a character sheet. Argument: $ARGUMENTS (character name).

1. Прочитай CLAUDE.md — найди персонажа в таблице
2. Прочитай ВСЕ файлы из scenes/ по порядку
3. Найди каждое упоминание персонажа: диалоги, описания действий, ремарки
4. Составь карточку:

```
## ИМЯ ПЕРСОНАЖА (возраст)

### Роль
Кто в истории: протагонист / антагонист / союзник / ...

### Первое появление
Сцена NN — как введён, что делает

### Сцены с участием
| Сцена | Что делает | С кем взаимодействует |
|-------|-----------|---------------------|

### Речевой портрет
- Типичные фразы
- Манера говорить (длинные/короткие реплики, жаргон, паузы)
- Обращения к другим

### Отличимость голоса
Сравни речь этого персонажа с другими: можно ли отличить его реплики без имени? Что делает его речь уникальной? Если речь неотличима от другого персонажа — отметь это как проблему.

### Отношения с другими персонажами
| Персонаж | Тип отношений | Динамика (растёт/падает/стабильна) |
|----------|---------------|------------------------------------|

### Арка
Откуда → куда движется персонаж по ходу сценария

### Внешность / реквизит
Все упоминания одежды, предметов, физических деталей (со ссылками на сцены)

### Проблемы
Противоречия, пропуски, сцены где персонаж упомянут но не работает на историю
```

5. Выведи карточку. Не сохраняй в файл — только покажи.

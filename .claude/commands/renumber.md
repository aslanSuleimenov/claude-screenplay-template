Renumber scenes/blocks in scenes/. Argument: $ARGUMENTS (optional — starting number or range).

## Шаг 1 — Прочитай состояние

1. Прочитай CLAUDE.md — определи тип проекта (художественный → «Сцена», документальный → «Блок»)
2. Найди все файлы в scenes/ по алфавиту
3. Выведи текущую нумерацию пользователю
4. Создай резервную копию: заархивируй папку scenes/ в `versions/backup_ГГГГММДД_ЧЧММ_перенумерация.zip` перед любыми изменениями

## Шаг 2 — Рассчитай новую нумерацию

- По умолчанию: последовательная нумерация 01, 02, 03...
- Если >99 файлов — трёхзначная: 001, 002, 003...
- Файл 00_* (титульная) — не трогать
- Если указан аргумент — начать с указанного номера

## Шаг 3 — Двухфазный rename

**Фаза A:** Переименуй все файлы во временные имена `temp_XXXX_название.md` (чтобы избежать коллизий на Windows).

**Фаза B:** Переименуй из временных в финальные `NN_название.md`.

## Шаг 4 — Обнови заголовки внутри файлов

В каждом файле замени заголовок:
- Художественный: `# Сцена XX: Название` → `# Сцена NN: Название`
- Документальный: `# Блок XX: Название` → `# Блок NN: Название`

## Шаг 5 — Обнови CLAUDE.md

- Перестрой таблицу сцен с новыми номерами и именами файлов
- Добавь запись в «Историю изменений»

## Шаг 6 — Итог

Выведи таблицу: старое имя → новое имя.

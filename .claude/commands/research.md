Research a topic for the screenplay. Argument: $ARGUMENTS (research topic).

1. Прочитай CLAUDE.md — жанр, место действия, контекст проекта
2. Поищи в интернете (WebSearch) по теме: факты, статистика, реальные случаи, визуальные детали
3. Если тема связана с профессией/средой — найди специфическую лексику, жаргон, типичные ситуации
4. Если тема связана с локацией — найди географические, культурные, бытовые детали

5. Сохрани результат в `analytics/research_[тема].md`:

```
## Исследование: [тема]

Дата: [сегодня]
Для проекта: [название]

### Факты
- ...

### Детали для сценария
(что можно использовать в диалогах, действии, реквизите)

### Источники
- [ссылки]
```

6. Скажи какие сцены могут выиграть от этих деталей.

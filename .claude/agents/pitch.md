Ты создаёшь инвестиционный питч-документ для сценарного проекта.

## Шаг 1: Сбор данных о проекте

Прочитай CLAUDE.md — название, тип (художественный/документальный), жанр, логлайн, персонажи, хронометраж, целевая аудитория.

Прочитай все файлы в scenes/ (по алфавиту) — извлеки:
- главных персонажей и их арки
- центральный конфликт
- ключевые повороты и финал
- тон, атмосфера, визуальный стиль

## Шаг 2: Жанровый контекст

Прочитай analytics/compass_artifact.md если есть — жанровые системы, эталонные фильмы, диагностика.

Затем определи жанр проекта и прочитай нужный файл из ../compass/:
- документальный портрет → ../compass/doc/portrait.md
- cinema verité → ../compass/doc/verite.md
- криминальный триллер → ../compass/fiction/thriller.md
- чёрная комедия → ../compass/fiction/black-comedy.md
- научная драма → ../compass/fiction/sci-drama.md

Из жанрового файла возьми: эталонные проекты, жанровые конвенции — для раздела «Референсные проекты».

## Шаг 3: Напиши pitch.md

Структура документа:

```
# Питч: [Название]

**Жанр:** ...
**Формат:** ... (хронометраж, тип — полный метр / докфильм / серии)
**Дата:** ...

---

## Логлайн

Одно-два предложения. Кто герой, какова ставка, в чём конфликт.
Формула: [Герой] должен [цель] несмотря на [препятствие] — иначе [последствие].

---

## Синопсис

~350 слов. Три части: завязка, развитие, финал.
Без спойлера финала — только обозначить ставку.
Пишешь так, чтобы инвестор понял суть за 2 минуты чтения.

---

## Главные персонажи

Для каждого главного героя (3–5):
**Имя** — одна строка кто это, одна строка в чём его внутренний конфликт.

---

## Референсные проекты

«[Название] — это встреча [Фильм A] и [Фильм B]».
2–3 референса с объяснением: что именно роднит (тон? структура? тема?).
Бери из compass_artifact и жанрового файла compass.

---

## Аудитория

Кто смотрит. Платформы. Возраст, интересы. 3–4 предложения.

---

## Почему сейчас

Актуальность темы. Культурный или социальный момент. 2–3 предложения.
Конкретно — без «это важная и актуальная тема».

---

## Бюджетный уровень

Низкий / средний / высокий + обоснование (локации, актёры, спецэффекты).
Аналог: какие известные фильмы снимались в похожем диапазоне.
```

## Правила написания

- Деловой, но не канцелярский тон
- Без AI-паттернов из analytics/avoid-ai-writing-tells.md
- Синопсис — настоящее время («Герой едет», не «Герой поедет»)
- Логлайн — без названия фильма внутри
- Никакого пафоса в описании темы — только конкретика

Сохрани результат в analytics/pitch.md.
Сообщи сколько сцен прочитал и какой жанровый файл использовал.
